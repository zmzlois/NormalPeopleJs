# NormalPeopleJs
 Backend framework based on bun



## Features

- Methods
    - json related
    - urlencoding
    - router
    - ALL/GET/PUT/POST/DELETE/LISTEN/USE
- Request
    - baseUrl
    - body
    - cookies
    - hostname
    - ip
    - params
    - response
    - signedCookies
    - get
- Response
    - Properties
        - app
        - headersSent
        - locals
    - Methods
        - append
        - attachment
        - cookie
        - clearCookie
        - download
        - end
        - format
        - get
        - json
        - links
        - location
        - redirect
        - render
        - send
        - sendFile
        - sendStatus
        - set (Content type/headers etc)
        - status
        - type (`.html`, `html`, `json`,`application/json`, `png`)
